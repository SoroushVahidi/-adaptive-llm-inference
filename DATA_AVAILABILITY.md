# Data Availability

This document describes the provenance, licensing, and access instructions for
all datasets used in the manuscript.

---

## 1. Public Benchmark Datasets

### GSM8K

- **Source:** HuggingFace — `openai/gsm8k`
- **License:** MIT
- **Access:** Auto-downloaded on first run via `datasets` library.  
  No manual download needed; results are cached in `~/.cache/huggingface/`.
- **Reference:** Cobbe et al., "Training Verifiers to Solve Math Word Problems," 2021.

### MATH500

- **Source:** HuggingFace — `lighteval/MATH-Hard` (500-problem subset)
- **License:** MIT (original MATH dataset: Hendrycks et al.)
- **Access:** Auto-downloaded on first run.
- **Reference:** Hendrycks et al., "Measuring Mathematical Problem Solving With the MATH Dataset," 2021.

### AIME-2024

- **Source:** 30 problems from the 2024 AMC/AIME competition.
- **Access:** A pre-processed version is committed at `data/real_aime2024_routing_dataset.csv`.
- **Note:** AIME problems are publicly available from the Art of Problem Solving (AoPS) community.

### GPQA-Diamond

- **Source:** Normalised subset committed at `data/gpqa_diamond_normalized.jsonl`.
- **Reference:** Rein et al., "GPQA: A Graduate-Level Google-Proof Q&A Benchmark," 2023.
- **Access:** Committed file; full dataset available at [https://github.com/idavidrein/gpqa](https://github.com/idavidrein/gpqa).

## 2. Derived Routing Datasets

The following datasets were generated by querying `gpt-4o-mini` (via the
OpenAI API) on subsets of the public benchmarks above.  They are committed
to `data/` for reproducibility.

| File | Queries | Model | Description |
|------|---------|-------|-------------|
| `data/real_gsm8k_routing_dataset.csv` | 100 | gpt-4o-mini | Paired outcomes: `reasoning_greedy` + `direct_plus_revise` |
| `data/real_gsm8k_routing_dataset_enriched.csv` | 100 | gpt-4o-mini | + query-level routing features |
| `data/real_math500_routing_dataset.csv` | 100 | gpt-4o-mini | MATH500 paired outcomes |
| `data/real_math500_routing_dataset_enriched.csv` | 100 | gpt-4o-mini | + routing features |
| `data/real_hard_gsm8k_routing_dataset.csv` | 100 | gpt-4o-mini | Hard-regime GSM8K paired outcomes |
| `data/real_hard_gsm8k_routing_dataset_enriched.csv` | 100 | gpt-4o-mini | + routing features |
| `data/real_hard_gsm8k_b2_routing_dataset.csv` | 100 | gpt-4o-mini | Hard GSM8K (second batch) |
| `data/real_hard_gsm8k_b2_routing_dataset_enriched.csv` | 100 | gpt-4o-mini | + routing features |
| `data/real_aime2024_routing_dataset.csv` | 30 | gpt-4o-mini | AIME-2024 routing outcomes |

### Column schema (enriched CSVs)

| Column | Description |
|--------|-------------|
| `question_id` | Unique identifier |
| `question` | Problem text |
| `gold_answer` | Ground-truth answer |
| `reasoning_answer` / `revise_answer` | Model outputs per strategy |
| `reasoning_correct` / `revise_correct` | Binary correctness labels |
| `revise_helpful` | 1 if revision corrected a reasoning error |
| `feat_*` | Query-level routing features (see `src/features/`) |

### Regeneration

To regenerate from scratch (requires `OPENAI_API_KEY`):

```bash
# GSM8K
python3 scripts/run_build_real_routing_dataset.py \
    --paired-outcomes --subset-size 100 \
    --output-dataset-csv data/real_gsm8k_routing_dataset.csv

# MATH500
python3 scripts/run_build_math500_routing_dataset.py --subset-size 100

# Hard GSM8K
python3 scripts/run_select_hard_gsm8k.py --subset-size 100
python3 scripts/run_build_hard_gsm8k_routing_dataset.py
```

## 4. What Is NOT Committed

- Full HuggingFace dataset downloads (`data/` is gitignored except for the
  files listed above).
- Model weights, checkpoints, or embeddings.
- Raw LLM response caches beyond the `raw_responses.jsonl` files committed
  under `outputs/`.

## 5. Licensing Note

The derived routing datasets inherit the licence of their source benchmarks
(GSM8K: MIT; MATH500: MIT; AIME: public competition problems).  The feature
extraction and routing labels are the authors' original work and are released
under the repository's MIT licence.
