# docs/internal/

This directory contains internal working notes, AI-agent planning documents,
and in-progress research logs that are **not intended for external readers**.

These files were kept in the repository for historical traceability during
active development but should be treated as internal scratch material:

- `CODEX_CRITICAL_STATE_REVIEW.md` — AI agent state review notes
- `CODEX_NEXT_EXPERIMENT_RECOMMENDATION.md` — AI agent experiment planning
- `TONIGHT_THINKING_QUESTION.md` — informal researcher notes
- `OPEN_QUESTIONS_NOW.md` — live open-questions log
- `DECISION_POINT_SUMMARY.md` — internal decision log
- `NEXT_REPO_RECOVERY_PLAN.md` — recovery planning note
- `FULL_REPO_AUDIT.md`, `FULL_PROJECT_STATE_AUDIT.md`, `REPO_AUDIT_REPORT.md` — audit logs
- `MAIN_BRANCH_STATUS_AUDIT.md`, `FINAL_PRE_RUN_AUDIT.md` — pre-run checks
- `PAPER_EXPORT_RUN_STATUS.md`, `PAPER_ARTIFACT_GENERATION_STATUS.md` — run status logs
- `CURRENT_STATE_SUMMARY.md` — work-in-progress state snapshot
