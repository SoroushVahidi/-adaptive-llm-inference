# Figure Provenance

This document maps each paper-ready figure to the scripts and input artifacts that produced it.
All figures are generated by `scripts/generate_lightweight_paper_artifacts.py`.

---

## routing_headroom_barplot.png

**Output path:** `outputs/paper_figures/routing_headroom_barplot.png`

**Description:** Grouped bar chart comparing Always-RG, Always-DPR, Best Adaptive, and Oracle
accuracy side-by-side for each regime. Shows the "headroom" available from better routing.

**Script function:** `_figure_headroom_barplot()` in `scripts/generate_lightweight_paper_artifacts.py`

**Input artifacts:**
- `outputs/real_math500_policy_eval/policy_comparison.csv`
- `outputs/real_hard_gsm8k_policy_eval/policy_comparison.csv`
- `outputs/real_hard_gsm8k_b2_policy_eval/policy_comparison.csv`
- `outputs/real_policy_eval/policy_comparison.csv`
- `outputs/oracle_routing_eval/math500_100_oracle_summary.json`
- `outputs/oracle_routing_eval/hard_gsm8k_100_oracle_summary.json`
- `outputs/oracle_routing_eval/hard_gsm8k_b2_oracle_summary.json`
- `outputs/oracle_routing_eval/gsm8k_random100_oracle_summary.json`

---

## adaptive_efficiency_scatter.png

**Output path:** `outputs/paper_figures/adaptive_efficiency_scatter.png`

**Description:** Scatter plot with average cost on the x-axis and accuracy on the y-axis.
Each point is a (policy, regime) pair. Shows the accuracy-cost frontier.

**Script function:** `_figure_efficiency_scatter()` in `scripts/generate_lightweight_paper_artifacts.py`

**Input artifacts:**
- `outputs/real_math500_policy_eval/policy_comparison.csv`
- `outputs/real_hard_gsm8k_policy_eval/policy_comparison.csv`
- `outputs/real_hard_gsm8k_b2_policy_eval/policy_comparison.csv`
- `outputs/real_policy_eval/policy_comparison.csv`

---

## oracle_gap_barplot.png

**Output path:** `outputs/paper_figures/oracle_gap_barplot.png`

**Description:** Bar chart showing the accuracy gap from (a) oracle to Always-RG and
(b) oracle to best adaptive policy, by regime. Illustrates remaining improvement potential.

**Script function:** `_figure_oracle_gap_barplot()` in `scripts/generate_lightweight_paper_artifacts.py`

**Input artifacts:**
- `outputs/real_*/policy_comparison.csv` (for RG and best adaptive accuracy)
- `outputs/oracle_routing_eval/*_oracle_summary.json` (for oracle accuracy)

---

## routing_outcome_stacked_bar.png

**Output path:** `outputs/paper_figures/routing_outcome_stacked_bar.png`

**Description:** Stacked bar chart showing the fraction of queries in each routing outcome
category (both correct / RG-only correct / DPR-only correct / both wrong) per regime.
The "DPR-only correct" fraction represents the theoretical routing headroom.

**Script function:** `_figure_routing_outcome_stacked_bar()` in `scripts/generate_lightweight_paper_artifacts.py`

**Input artifacts:**
- `outputs/paper_tables/routing_outcome_breakdown.csv` (generated in the same script run)
- Ultimately derived from:
  - `outputs/real_math500_policy_eval/per_query_policy_decisions.csv`
  - `outputs/real_hard_gsm8k_policy_eval/per_query_policy_decisions.csv`
  - `outputs/real_hard_gsm8k_b2_policy_eval/per_query_policy_decisions.csv`
  - `outputs/real_policy_eval/per_query_policy_decisions.csv`

---

## threshold_tradeoff_curve.png

**Output path:** `outputs/paper_figures/threshold_tradeoff_curve.png`

**Description:** Line plot of accuracy vs. average cost as the confidence threshold is swept,
one line per regime. Shows the trade-off curve for the threshold-based baseline.

**Script function:** `_figure_threshold_tradeoff_curve()` in `scripts/generate_lightweight_paper_artifacts.py`

**Input artifacts:**
- `outputs/baselines/confidence_threshold/confidence_threshold_sweep.csv`

---

## Regeneration

All figures can be regenerated with:

```bash
python scripts/generate_lightweight_paper_artifacts.py
```

The script is deterministic and idempotent.
