\noindent\textbf{Notes.}
\textit{Practical policy vs.\ budget frontier:}
The ``Best adaptive policy'' is a single trained routing policy with a fixed threshold,
deployable at inference time without oracle knowledge.
The ``Budget frontier'' values are from a sweep analysis that picks the revision fraction
retrospectively to match each target cost level---they show the Pareto-optimal accuracy
for that budget but are \emph{not} the same operating point as the practical policy.
\textit{Oracle:}
The oracle result requires knowing in hindsight whether revision helps each individual query;
it is an upper bound on what any deployable policy can achieve, not a practical system.
